## 1. 系统简介

### 1.1 系统概述

上海迪士尼舆情全自动分析系统是一个集成多种前沿技术的智能舆情分析平台，能够实现从数据采集到报告生成的全流程自动化。

### 1.2 核心功能

| 功能模块 | 描述 |
|---------|------|
| 🚀 数据爬取 | 支持微博、知乎、贴吧、虎扑四大平台 |
| 🤖 情感分析 | SnowNLP快速分析 + DeepSeek深度分析 |
| 🗺️ 知识图谱 | 实体-关系网络构建与分析 |
| 📡 舆情传播 | 影响力节点识别与传播路径分析 |
| ⏰ 情感演化 | 时间序列分析与趋势预测 |
| 🎨 多模态分析 | 文本+表情+图片综合情感分析 |
| 📊 可视化 | 交互式驾驶舱展示 |
| 📑 报告生成 | 自动生成学术分析报告 |

### 1.3 系统要求

- **操作系统**: Windows 10/11, macOS, Linux
- **Python版本**: 3.10+
- **内存**: 建议 8GB 以上
- **磁盘空间**: 5GB 以上

## 2. 快速开始

### 2.1 安装步骤

#### 步骤1: 克隆项目

```bash
git clone <repository-url>
cd bishe
```

#### 步骤2: 创建虚拟环境

```bash
# Windows
python -m venv .venv
.venv\Scripts\activate

# macOS/Linux
python3 -m venv .venv
source .venv/bin/activate
```

#### 步骤3: 安装依赖

```bash
pip install -r requirements.txt
```

#### 步骤4: 配置API密钥

编辑 `src/config/settings.py` 或设置环境变量：

```bash
# Windows
set DEEPSEEK_API_KEY=your-api-key-here

# macOS/Linux
export DEEPSEEK_API_KEY=your-api-key-here
```

### 2.2 启动系统

#### 方式1: 启动前端界面（推荐）

```bash
streamlit run app.py --server.port 8502
```

然后在浏览器中访问: http://localhost:8502

#### 方式2: 运行全流程分析

```bash
python run_full_analysis.py
```

## 3. 功能详解

### 3.1 驾驶舱

**功能描述**: 核心数据展示与可视化分析

**主要功能**:
- 📊 CSI满意度指数实时监控
- 😊 情感分布饼图
- 📈 满意度分布直方图
- ⚡ 紧急度分布分析
- 🏢 设施类型分析
- 📋 方面维度分析
- 🔬 深度分析（桑基图、词云等）

**使用方法**:
1. 点击左侧导航栏「驾驶舱」
2. 系统自动加载已分析的数据
3. 查看各类图表和统计数据

### 3.2 数据中心

**功能描述**: 数据上传、爬取与分析

**主要功能**:
- 📤 上传本地CSV数据
- 🕷️ 启动爬虫采集数据
- 🤖 运行情感分析
- ⚙️ 分析设置（选择分析模式）

**使用方法**:

#### 上传数据:
1. 点击「数据中心」
2. 在「上传数据」区域选择CSV文件
3. 点击「上传并预览」

#### 运行分析:
1. 选择分析模式:
   - **SnowNLP**: 快速分析，适合大数据量
   - **DeepSeek**: 深度分析，精度高
   - **混合模式**: 推荐，平衡速度与精度
2. 调节DeepSeek分析比例（5%-30%）
3. 点击「开始分析」
4. 等待分析完成

### 3.3 前沿技术

**功能描述**: 前沿技术分析模块

**包含模块**:

#### 🗺️ 知识图谱
- 实体分布统计
- Top实体排行
- 问题实体分析
- 优质实体分析

#### 📡 舆情传播
- 影响力节点排行
- 情感分布分析
- 时间序列趋势

#### ⏰ 情感演化
- 演化阶段检测
- 情感转移矩阵
- 新兴话题检测

#### 🎨 多模态分析
- 单条评论分析
- 表情符号识别
- 批量分析统计

### 3.4 智能问答

**功能描述**: 基于RAG的智能问答系统

**使用方法**:
1. 点击「智能问答」
2. 在输入框输入问题，例如：
   - "哪些设施的卫生问题最严重？"
   - "用户对服务态度的评价如何？"
   - "有什么改进建议？"
3. 系统自动检索相关数据并生成回答

### 3.5 报告下载

**功能描述**: 下载自动生成的分析报告

**可下载内容**:
- 📄 Markdown格式学术报告
- 📊 分析数据CSV文件
- 🖼️ 可视化图表文件

## 4. 操作指南

### 4.1 完整分析流程

```
步骤1: 数据采集
    ├── 方式A: 上传本地CSV文件
    └── 方式B: 启动爬虫采集

步骤2: 数据分析
    ├── 选择分析模式（推荐混合模式）
    ├── 调节DeepSeek比例（推荐10%）
    └── 点击「开始分析」

步骤3: 查看结果
    ├── 驾驶舱: 查看可视化图表
    ├── 前沿技术: 深度分析
    └── 智能问答: 交互式查询

步骤4: 导出报告
    └── 报告下载: 获取分析报告
```

### 4.2 爬虫使用指南

#### 微博爬虫

```bash
python run_weibo_only.py
```

**参数配置**:
- 关键词: 在脚本中修改 `KEYWORD` 变量
- 页数: 修改 `MAX_PAGES` 变量

#### 知乎爬虫

```bash
python run_zhihu_only.py
```

#### 贴吧爬虫

```bash
python run_tieba_only.py
```

#### 虎扑爬虫

```bash
python run_hupu_only.py
```

## 5. 常见问题

### Q1: 启动时提示模块找不到？

**解决方案**:
```bash
# 确保在项目根目录
cd bishe

# 激活虚拟环境
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # macOS/Linux

# 重新安装依赖
pip install -r requirements.txt
```

### Q2: 爬虫运行失败？

**可能原因**:
1. 网络连接问题
2. 目标网站反爬机制
3. 浏览器驱动版本不匹配

**解决方案**:
1. 检查网络连接
2. 更新Edge浏览器到最新版本
3. 使用断点续跑功能

### Q3: DeepSeek API调用失败？

**解决方案**:
1. 检查API密钥是否正确
2. 检查网络是否能访问API
3. 使用SnowNLP作为备选方案

### Q4: 分析速度太慢？

**优化建议**:
1. 使用混合模式，降低DeepSeek比例
2. 减少数据量，分批处理
3. 使用缓存机制

### Q5: 内存不足？

**解决方案**:
1. 减少批处理数据量
2. 关闭其他应用程序
3. 增加虚拟内存
